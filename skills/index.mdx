MNKY Atlas now acts as the docs registry for the current skill collection.

## How to use this library

- Read the human page first.
- Use the adjacent `skill.md` file as the machine-readable source for MCP and API retrieval.
- Treat the catalog as the index and the raw skill files as the install payload.
- If a skill changes, update the source `SKILL.md` first, then regenerate this registry.

## Machine-readable entrypoints

- Manifest: `/skills/manifest.json`
- Search surface: Mintlify search and MCP
- Skill payload: `/skills/<skill>/skill.md`
