MNKY Atlas is the curated docs layer for the MNKY agent ecosystem.

## How to use it
- Read the overview pages first.
- Use the host pages when you need deployment-specific detail.
- Keep durable notes in the Obsidian vault, and publish the polished version here.

## Source of truth
- Operational notes and working drafts live in Obsidian.
- Shared filesystem state lives on NFS.
- This repo is the curated docs surface.
